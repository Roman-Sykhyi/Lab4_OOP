# Лабораторна робота №4 - Наслідування класів
27. Створити базовий клас ДРУКОВАНА ПРОДУКЦІЯ (задаються назва, кількість сторінок). Створити похідний клас ГАЗЕТА (задаються назва, розмір сторінки, кількість примірників, періодичність виходу на тиждень). Для введених даних про друковану продукцію за тиждень обчислити витрати паперу (у кв. м.) – для кожного прикладу і сумарні, відсортувати ці приклади за зростанням витрат.
